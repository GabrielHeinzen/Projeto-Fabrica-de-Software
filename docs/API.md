Documentação completa disponível no Postman:

## Endpoints

### Autenticação
- POST /login

### Empresas
- GET /empresa
- POST /empresa
- PUT /empresa/:id
- DELETE /empresa/:id

### Documentos
- GET /documentos
- POST /documentos
- PUT /documentos/:id
- DELETE /documentos/:id

### Anexo de Documentos
- POST /empresa/:id/documentos
- GET /empresa/:id/documentos/status

### Dashboard
- GET /dashboard
